# readme-creator
npm Module for creating a README.md file based on asnwers to questions on terminal

## Install
```
npm install readme-creator
```

## Usage
1. Add a script key with `readme` to your project's `package.json` file. 
```
"scripts": {
  "start": "readme"
}
```

1. Run the added script.
```
npm start
```
